# Примеры данных

Эта папка содержит примеры данных для тестирования плагина SnapFill.

## Файлы

### JSON файлы

- **`test-data.json`** — простой JSON объект с базовыми данными
- **`test-data-array.json`** — массив JSON объектов для множественного заполнения
- **`dyson-example.json`** — пример с реальными данными о продуктах Dyson
- **`products-collection.json`** — коллекция различных товаров

### CSV файл

- **`test-data.csv`** — пример CSV файла с данными

## Использование

1. Откройте плагин SnapFill в Figma
2. Выберите один из файлов из этой папки
3. Загрузите его в плагин
4. Выберите фрейм(ы) и нажмите "Заполнить"

## Создание своих примеров

### JSON формат

```json
{
  "title": "Название товара",
  "description": "Описание товара",
  "price": "1000 ₽",
  "image": "https://example.com/image.jpg"
}
```

Или массив для множественного заполнения:

```json
[
  { "title": "Товар 1", "price": "100" },
  { "title": "Товар 2", "price": "200" }
]
```

### CSV формат

```csv
title,description,price,image
Товар 1,Описание 1,1000,https://example.com/1.jpg
Товар 2,Описание 2,2000,https://example.com/2.jpg
```

## Советы

- Называйте поля в данных так же, как слои в Figma
- Для изображений используйте поля с URL
- Убедитесь, что все записи имеют одинаковую структуру
