## 🚀 Как запустить

### 1. Загрузить плагин в Figma Desktop

```
Figma → Plugins → Development → Import plugin from manifest...
→ Выбрать файл: manifest.json
```

### 2. Подготовить тестовый макет

Создать Frame с текстовыми слоями:
- `title`
- `subtitle`
- `price`
- `description`
- `badge`
- `rating`

И Rectangle с именем:
- `image`

### 3. Запустить и протестировать

1. Выделить Frame
2. Plugins → Development → SnapFill
3. Загрузить test-data.json
4. Нажать "Заполнить"

**Подробнее:** см. INSTRUCTIONS.md

## 🎯 Примеры использования

```

├── Text "title"         → "iPhone 15 Pro"
├── Text "price"         → "от 99 990 ₽"
├── Text "badge"         → "Новинка"
└── Rectangle "image"    → загрузит изображение
```

### Профиль пользователя
```
Frame "UserProfile"
├── Text "name"          → "Александр Аверин"
├── Text "role"          → "Lead Product Designer"
├── Text "email"         → "email@example.com"
└── Rectangle "avatar"   → загрузит аватар
```
