# Руководство по пресетам

SnapFill включает встроенные пресеты данных для быстрого прототипирования макетов.

## Категории пресетов

### B2B Пресеты

Пресеты для бизнес-приложений и маркетплейсов:

#### 1. Каталог товаров
Список товаров с характеристиками и статусами.

**Поля:**
- `id` — ID товара
- `image` — изображение товара
- `title` — название товара
- `category` — категория
- `status` — статус ("Нет на складе", "В наличии")
- `quality` — качество карточки (число)
- `stock` — количество на складе
- `reserve` — количество в резерве
- `price` — цена
- `errors` — ошибки/предупреждения
- `service_cost` — стоимость услуг
- `discount` — скидка

#### 2. Цены товаров
Управление ценами и ценовыми предложениями.

**Поля:**
- `id`, `image`, `title`, `category`
- `price1`, `price2` — разные варианты цен
- `price_from` — цена "от"
- `status_badge` — "Скидка Маркета", "Завышенная цена"

#### 3. Буст продаж
Рекламные кампании для увеличения продаж.

**Поля:**
- `campaign_name` — название кампании
- `tags` — теги
- `status` — статус
- `products_count` — количество товаров
- `bid_rate` — ставка
- `impressions`, `clicks`, `cart_adds`, `ordered`, `delivered`
- `expenses`, `revenue` — финансовые показатели

#### 4. Буст показов
Кампании для увеличения видимости товаров.

**Поля:**
- Те же что в "Буст продаж" + `reach`, `ctr`, `frequency`

#### 5. Полки
Размещение товаров на рекламных полках.

**Поля:**
- `campaign_name`, `products_count`, `category`
- `status`, `tags`, `bid`, `budget`
- Метрики: `impressions`, `clicks`, `orders`, `conversion`

#### 6. Охватное продвижение
Кампании для максимального охвата аудитории.

**Поля:**
- `status_icon` — иконка статуса
- Остальные поля аналогичны другим кампаниям

#### 7. Отзывы за баллы
Кампании стимулирования отзывов покупателей.

**Поля:**
- `campaign_name`, `tag`, `status`
- `products_in_campaign` — товаров в кампании
- `reviews_target` — цель по отзывам
- `reviews_left` — осталось собрать
- `max_budget` — максимальный бюджет

### B2C Пресеты

Пресеты для потребительских приложений:

#### 1. Списки товаров
Карточки товаров маркетплейса для потребителей.

**Поля:**
- `image`, `title`, `price`, `brand`, `category`
- `rating`, `rating_count` — рейтинг и количество отзывов
- `badge` — бейдж со скидкой
- `buy_count` — "567К купили"
- `delivery_date` — "Сегодня", "Завтра"
- `seller_name` — название продавца
- `split_payment` — цена при оплате частями

#### 2. Детали товара
Подробная информация о товаре для страницы детализации.

**Поля:**
- Все поля из "Списки товаров" +
- `description` — краткое описание
- `full_description` — полное описание
- `specs` — характеристики
- `stock_status` — "В наличии"

#### 3. Отзывы покупателей
Отзывы пользователей с оценками.

**Поля:**
- `author_name`, `author_avatar`
- `rating` — оценка (1-5)
- `review_text` — текст отзыва
- `review_date` — дата отзыва
- `helpful_count` — "234 человека нашли это полезным"
- `images` — фотографии к отзыву
- `verified_purchase` — "Подтвержденная покупка"

#### 4. Профили пользователей
Профили для социальных и административных интерфейсов.

**Поля:**
- `name`, `email`, `avatar`
- `role` — "Администратор", "Покупатель", "VIP"
- `status` — "Онлайн", "Не в сети"
- `join_date` — дата регистрации
- `location` — местоположение
- `bio` — биография
- `stats` — "1 240 заказов, 89 отзывов"

## Использование пресетов

### Шаг 1: Подготовьте макет
Создайте слои в Figma с именами, соответствующими полям пресета.

Например, для пресета "Списки товаров" создайте слои:
- `image`
- `title`
- `price`
- `rating`
- `badge`

### Шаг 2: Выберите пресет
1. Откройте плагин
2. Выберите вкладку B2B или B2C
3. Кликните на нужный пресет

### Шаг 3: Заполните
1. Выделите фреймы для заполнения
2. Нажмите кнопку "Заполнить"

## Множественное заполнение

Если вы выделите несколько фреймов, каждый получит случайные данные из пресета без повторов (в пределах одной операции).

**Пример:**
- Выделите 5 карточек товаров
- Выберите пресет "Списки товаров" (8 записей)
- Нажмите "Заполнить"
- Каждая карточка получит уникальный товар

## Советы

✅ **Именование слоев** — используйте точные имена полей из пресета  
✅ **Вложенность** — плагин ищет слои на всех уровнях вложенности  
✅ **Изображения** — слои с URL автоматически заполняются изображениями  
✅ **Множественное заполнение** — работает с любым пресетом

## Создание своих данных

Если встроенных пресетов недостаточно, используйте карточку "Загрузить файл" для загрузки своих JSON или CSV файлов.

Примеры форматов см. в папке [`examples/`](../examples/).
