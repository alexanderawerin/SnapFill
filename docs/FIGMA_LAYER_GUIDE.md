# Руководство по именованию слоев для SnapFill

## Принцип работы

SnapFill заполняет слои в Figma на основе их имён. **Имя слоя = Имя поля в данных**.

Если в JSON/CSV есть поле `"price": "99 990"`, то текстовый слой должен называться `price`.

---

## Типы данных и слоев

### Текстовые слои

Текстовые слои (Text) заполняются строковыми значениями из данных.

| Слой в Figma | Поле в JSON | Результат |
|--------------|-------------|-----------|
| `title` | `"title": "iPhone 15 Pro"` | iPhone 15 Pro |
| `price` | `"price": "99 990 ₽"` | 99 990 ₽ |
| `rating` | `"rating": "4.9"` | 4.9 |

### Изображения

Слои для изображений (Rectangle, Frame, Ellipse) заполняются URL-адресами.

| Слой в Figma | Поле в JSON | Результат |
|--------------|-------------|-----------|
| `image` | `"image": "https://..."` | Изображение загружается |
| `avatar` | `"avatar": "https://..."` | Изображение загружается |

**Важно:** URL должен начинаться с `https://`.

---

## Пример JSON

```json
{
  "image": "https://images.unsplash.com/photo-123?w=800",
  "badge": "25%",
  "price": "92 308 ₽",
  "title": "Очиститель воздуха Dyson PH04",
  "brand": "DYSON",
  "rating": "4.7"
}
```

### Соответствующие слои в Figma

```
Frame "ProductCard"
├── Rectangle "image"     → загрузит изображение
├── Text "badge"          → "25%"
├── Text "price"          → "92 308 ₽"
├── Text "title"          → "Очиститель воздуха Dyson PH04"
├── Text "brand"          → "DYSON"
└── Text "rating"         → "4.7"
```

## Рекомендации по именованию

### ✅ Хорошие имена слоев

- `title`, `subtitle`, `description`
- `price`, `old_price`, `discount`
- `image`, `avatar`, `logo`
- `rating`, `reviews`, `stars`
- `badge`, `tag`, `label`
- `brand`, `seller`, `vendor`

### ❌ Плохие имена

- `Text`, `Label`, `Value` — слишком общие
- `Frame 1`, `Group 2` — неинформативные
- `Main Slot`, `Side Slot` — названия компонентов
- `Icon / Star`, `Icon / Basket` — иконки, не данные

---

## Вложенность

Плагин рекурсивно обходит все дочерние элементы фрейма. Не важно, на каком уровне вложенности находится слой — если его имя совпадает с полем в данных, он будет заполнен.

```
Frame "Card"
├── Frame "Header"
│   └── Text "title"      → заполнится
├── Frame "Content"
│   └── Text "description" → заполнится
└── Frame "Footer"
    └── Text "price"      → заполнится
```

---

## Работа с компонентами

### Инстансы компонентов

Выделите инстанс компонента и запустите плагин — он заполнит все слои внутри.

### Несколько одинаковых слоев

Если несколько слоев имеют одинаковое имя, все они будут заполнены одним значением.

---

## Чувствительность к регистру

Имена слоев **чувствительны к регистру**:
- `Title` ≠ `title` ≠ `TITLE`

Убедитесь, что имя слоя точно совпадает с ключом в JSON/CSV.

---

## Как проверить имена слоев

1. Выделите слой в Figma
2. Посмотрите в левой панели его название
3. Переименуйте через двойной клик или клавишу F2

---

## Частые вопросы

### Почему некоторые поля не заполняются?
Проверьте, что имя слоя точно совпадает с ключом в JSON (регистр важен!)

### Можно ли заполнять вложенные фреймы?
Да! Плагин рекурсивно обходит все дочерние элементы.

### Как заполнить несколько текстов одним значением?
Дайте им одинаковые имена — все заполнятся одним значением.

### Работает ли с компонентами?
Да! Выделите инстанс компонента и запустите плагин.
