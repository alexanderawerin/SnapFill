# Changelog

Все изменения в проекте MDS • SnapFill.

## [Unreleased]

### Добавлено
- Поддержка Figma SECTION — теперь можно выбирать и заполнять секции наравне с фреймами, компонентами и инстансами

### Изменено
- **Bottom-up детекция карточек** — теперь плагин ищет помеченные поля (image, title, priceValue и т.д.) и от них строит понимание о карточках, вместо угадывания структуры сверху вниз. Работает с любой структурой файла.

## [0.0.2] — 2025-11-16

### Добавлено
- **Множественное заполнение** — выбор нескольких фреймов и заполнение случайными данными
- Режим Cards и Table для разных сценариев заполнения
- Коллекция тестовых товаров
- Документация по функциям

### Изменено
- Полностью переработан UI на React + shadcn/ui
- Компактный интерфейс с поддержкой тёмной темы
- Улучшены отступы и типографика

### Исправлено
- Исправлена загрузка изображений из URL
- Исправлен формат manifest.json для Figma API 1.0.0
- Очищены предупреждения консоли
